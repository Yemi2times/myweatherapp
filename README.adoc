= Simple weather app

== Getting started


You can execute the code using:

```
mvn clean package [ to compile, test and package the code ] 
mvn spring-boot:run -Dapp.weather.locations=Nigeria/Lagos,Nigeria/Kano [ to execute the code ]
```
